# RFID-Cloner-Replay

RDM6300 : TX D6, RX D8
LED : D5
RF Coil : D1
OLED : SDA D4, SCL D3

Original video : https://www.youtube.com/watch?v=uiY5ceqkfJw&t=183s
Android app: https://bit.ly/3q735l4 or you can go to Play Store and search for Talk to EPS32

Material:
ESP8266
RDM6300
OLED 0.96
Bluetooth HC-06 (must use HC-06 to work), HC-05 is not compatible

Note:
1. When uploading ESP8226 you need to take off Bluetooth HC-06 avoiding conflict
2. When you want to emulate/Replay the ID you need to move the RF Coil that attached to RDM6300 to PIN D1 and press Replay the put the coil on RFID rader
